## Instructions d'exécution

1. Utiliser Java 17.
2. Avoir un serveur MySQL local actif.
3. Créer une base de données nommée `petstore`.
4. Renseigner la connexion avec les variables d'environnement `PETSTORE_DB_URL`, `PETSTORE_DB_USER`, `PETSTORE_DB_PASSWORD` et `PETSTORE_DB_DRIVER`, ou avec les propriétés système `petstore.db.url`, `petstore.db.user`, `petstore.db.password` et `petstore.db.driver`.
5. Lancer la méthode `main` du fichier `fr.esaip.petstore.Main`.

## Tests

1. Exécuter `mvn test`.
